## 📋 Overview

**PromptForge** is a modern, responsive landing page for an AI prompt engineering studio. The website helps users discover, create, and optimize prompts for various AI models including ChatGPT, Claude, and Gemini. It features an interactive prompt generator that creates customized letter prompts based on user preferences.

![PromptForge Screenshot](https://via.placeholder.com/1200x630/8b5cf6/ffffff?text=PromptForge+Landing+Page)

## ✨ Features

- **Interactive Prompt Generator** - Real-time prompt creation for different letter types
- **Responsive Design** - Fully responsive layout that works on all devices
- **Clean UI/UX** - Modern, intuitive interface with Bootstrap 5
- **Template Library** - 200+ pre-built prompt templates
- **Copy Functionality** - One-click copy for generated prompts
- **Model Compatibility** - Works with ChatGPT, Claude, and Gemini
